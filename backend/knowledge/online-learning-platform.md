# Online Learning Platform - Knowledge Base

## Platform Overview

This is an Online Learning Platform designed for students, instructors, and administrators. The platform provides comprehensive course management, interactive lessons, quizzes, progress tracking, and certification features.

## User Roles

### Students
Students can:
- Browse and enroll in courses
- Access course lessons and materials
- Take quizzes to test their knowledge
- Track their learning progress
- Download certificates upon course completion
- View their dashboard with enrolled courses

### Instructors
Instructors can:
- Create and manage courses
- Add lessons to courses (videos, PDFs, external links)
- Create quizzes with multiple choice questions
- View analytics and student progress
- Manage course content and materials

### Administrators
Administrators can:
- Manage all users (students, instructors, admins)
- View platform-wide analytics
- Manage all courses and content
- Issue certificates to students
- Control platform settings including chatbot visibility

## Enrollment Process

### How to Enroll in a Course (Students)

1. **Browse Courses**: Navigate to the Courses page to see all available published courses
2. **View Course Details**: Click on a course to see its description, lessons, and quizzes
3. **Enroll**: Click the "Enroll" button on the course details page
4. **Access Course**: Once enrolled, the course appears in your Student Dashboard
5. **Start Learning**: Access lessons and quizzes from your enrolled courses

### Enrollment Status

- **Active**: Student is currently enrolled and learning
- **Completed**: Student has finished all lessons and quizzes
- **Inactive**: Enrollment is paused or cancelled

## Lessons

### How Lessons Work

1. **Lesson Types**:
   - Video lessons with progress tracking
   - PDF documents
   - External URL links
   - Text-based content

2. **Accessing Lessons**:
   - Navigate to your enrolled course
   - Click on a lesson to open it
   - For videos: Watch and your progress is automatically saved
   - For PDFs: Download or view in browser
   - For external links: Click to open in new tab

3. **Lesson Completion**:
   - Lessons are marked as complete when you finish viewing them
   - Video progress is tracked (you can resume from where you left off)
   - Completed lessons are shown with a checkmark

### Video Progress Tracking

- Your video watch progress is automatically saved
- You can pause and resume videos from where you left off
- Progress is synced across devices when logged in

## Quizzes

### How Quizzes Work

1. **Taking a Quiz**:
   - Navigate to the quiz from your course
   - Click "Take Quiz" to start
   - Answer all questions (multiple choice)
   - Submit your answers when complete

2. **Quiz Results**:
   - You'll see your score immediately after submission
   - Passing score is typically 70% or higher (varies by course)
   - You can review which answers were correct/incorrect
   - Quiz attempts are saved in your progress

3. **Quiz Types**:
   - Multiple choice questions
   - Single correct answer per question
   - Questions may have multiple correct answers (select all that apply)

### Quiz Attempts

- Each quiz can be taken multiple times
- Your best score is typically used for course completion
- All attempts are recorded in your quiz history
- Passing a quiz contributes to course completion

## Adding Lessons (Instructors)

### How to Add a Lesson

1. **Select Course**: Go to your course details page
2. **Click "Add Lesson"** or navigate to Lessons section
3. **Fill Lesson Details**:
   - Lesson Title (required)
   - Lesson Content/Description
   - Video URL (optional) - for video lessons
   - PDF URL (optional) - for PDF documents
   - External URL (optional) - for external links
   - Lesson Order/Position
4. **Save Lesson**: Click "Save" to add the lesson

### Lesson Types

- **Video Lesson**: Provide a video URL (YouTube, Vimeo, etc.)
- **PDF Lesson**: Upload or provide PDF document URL
- **External Link**: Link to external learning resources
- **Text Content**: Rich text content within the platform

## Adding Quizzes (Instructors)

### How to Add a Quiz

1. **Select Course**: Go to your course details page
2. **Navigate to Quizzes Section**
3. **Click "Create Quiz"**
4. **Fill Quiz Details**:
   - Quiz Title (required)
   - Description
   - Passing Score (percentage)
   - Link to Lesson (optional)
5. **Add Questions**:
   - Click "Add Question"
   - Enter question text
   - Select question type (Multiple Choice)
   - Add answer options
   - Mark correct answer(s)
   - Save question
6. **Save Quiz**: Click "Save Quiz" to create

### Question Types

- **Multiple Choice**: Single correct answer
- **Multiple Select**: Multiple correct answers (select all that apply)

## Progress Tracking

### How Progress Works

1. **Lesson Progress**:
   - Each completed lesson contributes to course progress
   - Video watch time is tracked
   - Progress percentage is calculated automatically

2. **Quiz Progress**:
   - Passing quizzes contributes to completion
   - Quiz attempts and scores are recorded
   - Best scores are used for progress calculation

3. **Course Completion**:
   - Complete all lessons in a course
   - Pass all required quizzes
   - Enrollment status changes to "Completed"
   - Become eligible for certificate

### Viewing Progress

- Student Dashboard shows overall progress
- Course details show per-course progress
- Progress page shows detailed breakdown
- Certificates page shows completed courses

## Certificates

### How Certificates Work

1. **Eligibility**:
   - Complete all lessons in a course
   - Pass all quizzes
   - Enrollment status must be "Completed"

2. **Issuance**:
   - Admin issues certificates to eligible students
   - Each certificate has a unique verification code
   - Certificate includes student name, course name, completion date

3. **Verification**:
   - Certificates can be verified using the verification code
   - Download certificate as PDF
   - Share certificate with employers or on LinkedIn

## Common Issues and Solutions

### Video Not Working

**Possible Causes:**
1. Video URL is invalid or expired
2. Browser compatibility issues
3. Network connectivity problems
4. Video platform restrictions

**Solutions:**
1. Check if video URL is correct and accessible
2. Try a different browser (Chrome, Firefox, Edge)
3. Check your internet connection
4. Clear browser cache and cookies
5. Contact instructor if video link needs updating

### Quiz Not Submitting

**Possible Causes:**
1. Not all questions answered
2. Network connectivity issues
3. Browser session expired

**Solutions:**
1. Ensure all questions have answers selected
2. Check internet connection
3. Refresh page and try again
4. Log out and log back in if session expired

### Progress Not Updating

**Possible Causes:**
1. Lesson not fully completed
2. Quiz not passed
3. Browser cache issues

**Solutions:**
1. Ensure you've watched entire video or completed lesson
2. Pass the quiz with required score
3. Refresh the page
4. Clear browser cache
